# 🛠️ TZT Patcher

> [!WARNING]
> **Use at your own risk.**

A runtime tweaking utility for the game on Linux.

## ✨ Features

- **Camera distance** — dynamically adjust the camera distance in real time
- **Weather** — switch any weather effect (including default) on the fly
- **River** — set any river vial type (0–7)
- **Particles (FoW)** — reveal enemy particles through the Fog of War
- **Game Plus** — enable Game Plus client-side (applied on game launch)
- **Launch & attach** — launch the game via Steam and automatically apply patches on startup

---

## 🚀 Build and Run

To compile and execute the patcher, run the following commands in your terminal:

```bash
# 1. Clone the repository
git clone https://github.com/UwUOcha/TZT-patcher-linux.git

# 2. Navigate into the directory
cd TZT-patcher-linux/

# 3. Build the project
make

# 4. Run the patcher (Root privileges are required)
./build/tzt_patcher
```
