# 🛠️ TZT Patcher

> [!WARNING]
> **Используйте на свой страх и риск.**

Утилита для изменения игровых параметров в реальном времени на Linux.

## ✨ Возможности

- **Дистанция камеры** — динамическая регулировка расстояния камеры в реальном времени
- **Погода** — переключение любой погоды (включая стандартную) на лету
- **Река** — установка любого типа русла реки (0–7)
- **Частицы (Туман войны)** — раскрытие вражеских частиц сквозь туман войны
- **Game Plus** — включение Game Plus на стороне клиента (применяется при запуске игры)
- **Запуск и внедрение** — запуск игры через Steam и автоматическое применение патчей при старте

---

## 🚀 Сборка и запуск

Для компиляции и запуска патчера выполните следующие команды в терминале:

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/UwUOcha/TZT-patcher-linux.git

# 2. Перейдите в директорию
cd TZT-patcher-linux/

# 3. Соберите проект
make

# 4. Запустите патчер (требуются права root)
./build/tzt_patcher
```
